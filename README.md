# OIBSIP_PYTHON_TAK_2-BMI-
This is a simple and interactive BMI Calculator developed using Python. The program takes the user's weight and height as input and calculates the Body Mass Index (BMI) using a standard formula. Based on the calculated value, it categorizes the user into different health conditions such as Underweight, Normal weight, Overweight, or Obese.
##Project Link

 https://www.programiz.com/online-compiler/3tLUKS7FIMcUM

 
## Objective
To calculate BMI using user input
To classify health condition based on BMI value
To implement basic Python concepts like functions and conditions
To create a beginner-friendly real-world application
## Steps Performed
Took input from the user (weight in kg and height in meters)
Applied the BMI formula
Calculated BMI value
Used conditional statements to classify BMI
Displayed result with category
Added loop to allow multiple calculations

## Tools Used
Python (Programming Language)
Programiz Online Compiler (for execution)
 Formula Used

BMI = Weight (kg) / Height² (m²)

## Outcome
Successfully developed a working BMI Calculator
Learned how to take user input and process data
Implemented conditional logic in Python
Created a simple real-life utility application
## How to Run
Open the project link
Click on Run
Enter your weight and height
View the BMI result instantly

## Conclusion

This project demonstrates how Python can be used to build simple and useful applications. It is a beginner-level project that helps in understanding core programming concepts.
